# 매니페스트 검증

## 목적

볼트 내 설정 매니페스트 파일(`_system/obsidian/`에 위치)과 실제 `.obsidian/` 설정을 비교하여 불일치를 보고한다.

## 절차

### 1. 매니페스트 로드

`_system/obsidian/` 아래에서 매니페스트 파일(볼트설정 관련)을 찾아 읽는다.
매니페스트가 없으면 "매니페스트 파일이 아직 생성되지 않았습니다"를 보고하고 종료한다.

### 2. 실제 설정 로드

`.obsidian/` 디렉토리에서 다음 파일을 읽는다:

| 파일 | 내용 |
|------|------|
| `app.json` | 핵심 설정 (새 파일 위치, 첨부 위치 등) |
| `appearance.json` | 테마, 폰트 |
| `core-plugins.json` | 코어 플러그인 ON/OFF |
| `community-plugins.json` | 커뮤니티 플러그인 목록 |
| `hotkeys.json` | 커스텀 단축키 |

### 3. 비교

매니페스트에 선언된 각 항목과 실제 설정을 비교한다:

| 항목 | 매니페스트 | 실제 | 상태 |
|------|----------|------|------|
| 새 파일 위치 | 06_inbox/ | 06_inbox/ | 일치 |
| Dataview 플러그인 | ON | ON | 일치 |
| Tasks 플러그인 | OFF | ON | **불일치** |

### 4. 보고서 출력

```
## 매니페스트 검증 보고

### 요약
- 선언 항목: N개
- 일치: N개
- 불일치: N개
- 매니페스트에만 있음: N개 (실제 미설정)
- 실제에만 있음: N개 (매니페스트 미선언)

### 불일치 상세
| 항목 | 매니페스트 | 실제 | 권장 조치 |
|------|----------|------|----------|
```

## 주의사항

- 이 스킬은 보고만 한다. `.obsidian/` 파일을 직접 수정하지 않는다.
- `workspace.json`, `graph.json`은 개인 설정이므로 비교 대상에서 제외한다.
