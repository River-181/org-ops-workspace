# 진행 상황 갱신

## 목적

현재 세션에서 수행한 PRJ-볼트v3 작업을 `PROGRESS.md`에 반영한다.

## 절차

### 1. 현재 상태 확인

`03_projects/PRJ-볼트v3/PROGRESS.md`를 읽어 현재 상태를 파악한다.

### 2. 변경사항 수집

이번 세션에서 수행한 작업을 정리한다:
- 어떤 Phase의 어떤 작업을 했는가?
- 결과는 무엇인가? (성공 / 부분 완료 / 블로커 발견)
- 새로운 미결 사항이 있는가?

### 3. PROGRESS.md 갱신

#### 상태 테이블 업데이트

해당 작업의 상태를 변경한다:

| 상태 | 의미 |
|------|------|
| `todo` | 아직 시작하지 않음 |
| `in-progress` | 진행 중 |
| `done` | 완료 |
| `blocked` | 미결 사항으로 진행 불가 |

#### 작업 로그 추가

작업 로그 섹션의 **최상단**에 오늘 날짜로 항목을 추가한다:

```markdown
### YYYY-MM-DD
- {수행한 작업 요약}
- {생성/수정된 파일이 있으면 목록}
```

### 4. PLAN.md 갱신 (해당 시)

새로운 미결 사항이 발견되면 `PLAN.md`의 미결 사항 섹션에 추가한다.
결정된 미결 사항은 제거하고 해당 Phase에 반영한다.

## 주의사항

- 현재 Phase의 모든 작업이 `done`이면 "현재 Phase: N" 표시를 다음 Phase로 변경한다.
- 마지막 업데이트 날짜를 갱신한다.
