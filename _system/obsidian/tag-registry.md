# Tag Registry

> [볼트명] 볼트의 공식 태그 레지스트리.
> `tag-register` 스킬이 새 태그를 등록한다. `tag-audit` 스킬이 미등록 태그를 감지한다.
>
> - 새 태그: 해당 네임스페이스 섹션에 알파벳순 삽입
> - 네임스페이스 신규 추가: `update-governance` 스킬로 CLAUDE.md도 함께 갱신
