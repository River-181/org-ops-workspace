# 볼트설정 매니페스트

> 이 볼트의 Obsidian 설정을 선언한 문서.
> 실제 설정은 `.obsidian/`에 있고, 이 파일은 그것의 **인간 가독 요약**이다.
> 동기화 방식: **수동** — 설정 변경 시 이 파일도 함께 업데이트한다.

---

## 테마

| 항목 | 값 |
|------|-----|

| 텍스트 폰트 | 기본 |
| 모노스페이스 폰트 | 기본 |

## 코어 플러그인 (권장 활성화)

| 플러그인 | 설명 |
|---------|------|
| file-explorer | 파일 탐색기 |
| global-search | 전체 검색 |
| switcher | 빠른 파일 전환 |
| graph | 그래프 뷰 |
| backlink | 역방향 링크 |
| tag-pane | 태그 패널 |
| properties | 속성(프론트매터) |
| page-preview | 페이지 미리보기 |
| templates | 템플릿 |
| command-palette | 명령 팔레트 |
| bookmarks | 북마크 |
| outline | 아웃라인 |
| bases | Bases (데이터베이스 뷰) |

## Git 추적 정책

`.obsidian/` 폴더는 전체 git 추적. 단, 아래 파일은 머신별 로컬 상태이므로 `.gitignore`로 제외:

```
.obsidian/workspace.json
.obsidian/workspace-mobile.json
.obsidian/graph.json
```
