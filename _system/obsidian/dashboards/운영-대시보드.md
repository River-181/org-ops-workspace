# 운영 대시보드

## 최근 운영 문서

```dataview
TABLE status, updated, file.folder
FROM "01_ops"
WHERE updated
SORT updated DESC
LIMIT 15
```

## draft 회의/모집/리뷰

```dataview
TABLE status, created, file.folder
FROM "01_ops"
WHERE status = "draft"
SORT created DESC
```

## 세션 plan / record

```dataview
TABLE status, created, updated
FROM "02_sessions"
WHERE contains(file.name, "plan") OR contains(file.name, "record")
SORT file.path ASC
```

## 미완료 task

```dataview
TASK
FROM "01_ops" OR "02_sessions" OR "03_projects"
WHERE !completed
SORT file.name ASC
```
