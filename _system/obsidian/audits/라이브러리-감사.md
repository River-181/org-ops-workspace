# 라이브러리 감사

> `05_library/`의 규모, 분포, 누락을 점검하는 감사 노트.
> `MOC-라이브러리`와 `MAP-라이브러리` 본문은 큐레이션용이며, 완전 목록과 부모 분포는 여기서 확인한다.

## MAP-라이브러리 직속 문서

```dataview
TABLE file.link AS "문서", kind, status, file.folder AS "폴더"
FROM "05_library"
WHERE file.ext = "md" AND up = [[MAP-라이브러리]]
SORT file.path ASC
```

## kind 분포

```dataview
TABLE WITHOUT ID kind AS "kind", length(rows) AS "문서 수"
FROM "05_library"
WHERE file.ext = "md"
GROUP BY kind
SORT length(rows) DESC
```

## 부모 분포

```dataview
TABLE WITHOUT ID up AS "부모", length(rows) AS "문서 수"
FROM "05_library"
WHERE file.ext = "md" AND up
GROUP BY up
SORT length(rows) DESC
```

## 폴더별 밀도

```dataview
TABLE WITHOUT ID file.folder AS "폴더", length(rows) AS "문서 수"
FROM "05_library"
WHERE file.ext = "md"
GROUP BY file.folder
SORT length(rows) DESC
```

## source_type 분포

```dataview
TABLE WITHOUT ID source_type AS "source_type", length(rows) AS "문서 수"
```


## 고아 후보

```dataview
TABLE file.link AS "문서", kind, file.folder AS "폴더", length(file.inlinks) AS "역링크 수"
FROM "05_library"
WHERE file.ext = "md" AND file.name != "README" AND length(file.inlinks) = 0
SORT file.path ASC
```
