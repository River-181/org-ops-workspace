# 볼트 감사

> 볼트 전체의 규모, 메타데이터 누락, 구조적 이상 징후를 점검하는 감사 노트.
> 운영용 허브는 `MOC`와 `MAP`, 전수 점검은 감사 노트가 맡는다.

## kind 분포

```dataview
TABLE WITHOUT ID kind AS "kind", length(rows) AS "문서 수"
FROM ""
WHERE file.ext = "md" AND !contains(file.path, "05_library/seasons/")
GROUP BY kind
SORT length(rows) DESC
```

## 폴더별 밀도

```dataview
TABLE WITHOUT ID file.folder AS "폴더", length(rows) AS "문서 수"
FROM ""
WHERE file.ext = "md" AND !contains(file.path, "05_library/seasons/")
GROUP BY file.folder
SORT length(rows) DESC
LIMIT 50
```

## 부모 누락

```dataview
TABLE file.link AS "문서", kind, file.folder AS "폴더"
FROM ""
WHERE file.ext = "md" AND !contains(file.path, "05_library/seasons/") AND !up AND file.name != "MOC-홈"
SORT file.path ASC
```

## kind 누락

```dataview
TABLE file.link AS "문서", status, file.folder AS "폴더"
FROM ""
WHERE file.ext = "md" AND !contains(file.path, "05_library/seasons/") AND !kind
SORT file.path ASC
```

## status 누락

```dataview
TABLE file.link AS "문서", kind, file.folder AS "폴더"
FROM ""
WHERE file.ext = "md" AND !contains(file.path, "05_library/seasons/") AND !status
SORT file.path ASC
```

## 최근 수정

```dataview
TABLE file.link AS "문서", kind, status, updated
FROM ""
WHERE file.ext = "md" AND updated AND !contains(file.path, "05_library/seasons/")
SORT updated DESC
LIMIT 50
```
