# [볼트명] 볼트설정 매니페스트

> 이 볼트의 Obsidian 설정을 선언한 문서.
> 실제 설정은 `.obsidian/`에 있고, 이 파일은 그것의 **인간 가독 요약**이다.
> 동기화 방식: **수동** — 설정 변경 시 이 파일도 함께 업데이트한다.

## Git 추적 정책

`.obsidian/` 폴더는 **전체 git 추적**. 단, 아래 파일은 머신별 로컬 상태이므로 `.gitignore`로 제외:

```
.obsidian/workspace.json          # 창 레이아웃 (로컬)
.obsidian/workspace-mobile.json   # 모바일 레이아웃 (로컬)
.obsidian/graph.json              # 그래프 뷰 설정 (로컬)
```
