> CLAUDE.md에서 분리된 스킬 상세 카탈로그. 스킬 description은 Claude Code가 자동 로드하므로, 이 파일은 사람이 전체 스킬을 훑어볼 때 참조한다.

## 세션 운영

| 스킬 | 설명 |
|------|------|
| `session-ops` | 세션 수명주기 통합 — `setup`(폴더 생성) / `launch`(준비 점검) / `record`(기록 완성) / `guard`(에이전트 세션) |
| `session-package` | 세션 자료 멤버 공유 폴더 패키징 — 공유 폴더 생성, 파일 복사, 네이밍 통일, README 작성 (세션 전·중·후 언제든) |
| `session-transcript` | 녹음본 2-track 변환 — 정제 스크립트(오염 제거, 원문 보존) + 세션 정리(섹션별 요약) |
| `session-close` | 세션 종료 마무리 — 폴더 파일 감사 + plan archived + 허브 session_status done + MOC-세션 링크 갱신 + 메모리 갱신 |
| `project-close` | 프로젝트(PRJ-*) 종료 마무리 — PROGRESS 최종화 + 허브 project_status/status 정합 + CLAUDE.md 참조 갱신 + 메모리 갱신 |
| `session-thread-open` | 세션 당일 Discord 스레드 오프닝 메시지 전송 — plan + 허브에서 자동 추출 → 봇 전송 |
| `session-announce` | 세션 공지 자동화 — 포스터 PNG 캡처 + `#announcements` 초안을 발행 직전까지 준비 |
| `session-research` | 세션 발표 재료 리서치 — plan 읽기 → 주제별 리서치 → `materials/` 저장 |
| `session-review` | 세션 후 운영팀 리뷰 — 목표 달성 점검 → ops 리뷰 초안 → 다음 세션 힌트 |
| `weekly-review` | 볼트 전체 스캔 → 현황 보고 생성 |
