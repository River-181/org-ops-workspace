# org-ops-workspace

> 소규모 조직을 위한 로컬 퍼스트 운영 하네스
> A local-first operations harness for small organizations

**지원 조직 유형:** 클럽 · 동아리 · 스타트업 · 팀플 · 학생회 · 하위팀 · 기타

---

## 빠른 시작

1. 이 레포를 "Use this template"으로 복제하거나 `git clone`
2. Obsidian에서 볼트로 열기
3. Claude Code에서 열어 `/org-init` 실행
4. 질문에 답하거나 기존 문서를 제공하면 자동으로 세팅 완료

> Obsidian만 사용한다면: `_system/identity/SETUP-GUIDE.md` 참고
